Mis materias

Descripción
Este curso atiende a la necesidad de la industria de desarrollo de software de aplicacion, de generar la mayor cantidad de productos de software en el menor tiempo posible, aprovechando las tecnologías de información y comunicación que permitan a equipos de trabajo geograficamente dispersos colaborar en un mismo proyecto de desarrollo. Para ello propone una metodología de programación denominada Programación Orientada a Objetos que permite reducir los tiempos de codificación a través de la descomposición del programa en componentes de software que sean fáciles de comprender, integrar, reutilizar y modificar/extender su funcionalidad.

Descripción
La materia tiene como finalidad conocer los principios de la arquitectura, programación y desarrollo de los microprocesadores; para ello se emplea, a manera de ejemplo, un microcontrolador de la familia 8051 en modo microprocesador. Para el desarrollo de esta materia es necesario el tener presentes los conceptos de la materia de fundamentos de sistemas digitales, particularmente el diseño mediante el uso de bloques funcionales y el uso de sistemas numéricos y su correspondiente aritmética.

Descripción
Los usuarios de las aplicaciones web interactuan a través de una interface gráfica que permite capturar y validar la información a ser almacenada en el servidor o bien desplegar la informacion recibida de este. El servidor es responsable de procesar la de información, administrar las bases de datos, administrar los repositorios de archivos, autentificar a los usuarios, entre otros. El servidor debe procesar la petición del cliente y enviar la respuesta solicitada. En esta asignatura harás uso de tus conocimientos de programación orientada a objetos y del diseño de bases de datos para crear proyectos más robustos, ya que conocerás y emplearás las principales herramientas y frameworks de desarrollo de aplicaciones web. Podrás desarrollar tus propias aplicaciones y servicios web en donde se administren los recursos del servidor. Además, podrás desarrollar tus propias páginas web interactivas, permitiendo que el usuario almacene, visualice y modifique la información de una base de datos.
Créditos 8
Descripción
Este curso pretende complementar a la materia Programación orientada a objetos al aplicar dicha metodología para construir aplicaciones de software robustas, utilizando dos lenguajes orientados a objetos y metodologías de actualidad. Dichas aplicaciones deben contar con una interfaz gráfica de usuario para que los usuarios puedan capturar, modificar y visualizar información proveniente de una base de datos local o remota, de un conjunto de archivos o de un servicio web; asimismo, la información podrá estar organizada en memoria para efectuar consultas eficientes. También se podrán crear interfaces gráficas 2D para fines de entretenimiento, enseñanza o como herramienta CAD. Las aplicaciones deberán atender a los diferentes errores de captura de los usuarios.
